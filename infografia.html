¿Qué es una vista?

Tabla virtual

Una vista es una consulta SQL almacenada que se comporta como una tabla. No guarda datos físicamente; muestra resultados calculados en tiempo real a partir de las tablas base.

¿Para qué sirve?

Simplifica el acceso a información compleja, mejora la reutilización del código SQL, protege datos sensibles y hace que las consultas sean más legibles y mantenibles.

Conclusiones

✓

Aprendizajes del taller

  • El modelado relacional (conceptual → lógico → físico) es esencial para representar sistemas reales correctamente.
  • Las claves primarias y foráneas garantizan la integridad referencial y evitan redundancias en los datos.
  • Las vistas encapsulan consultas complejas con JOIN, facilitando su reutilización y mejorando la legibilidad.
  • Técnicas como LEFT JOIN + IS NULL y HAVING permiten detectar registros sin relación, muy útiles para auditorías.
  • Se recomienda usar vistas en consultas frecuentes y documentar bien el sistema para su mantenimiento futuro.
