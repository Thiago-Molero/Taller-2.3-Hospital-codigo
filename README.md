Base de Datos Clínica - Oracle

Proyecto de base de datos relacional desarrollado en Oracle SQL para gestionar información de una clínica.

Contenido

* Tablas (pacientes, médicos, especialidades, enfermedades, diagnósticos)
* Inserción de datos
* Consultas con JOIN
* Vistas (VIEW)

Tecnologías

* Oracle SQL
* GitHub

Ejecución

Ejecutar el script en Oracle en el siguiente orden:

1. Tablas
2. Datos
3. Consultas
4. Vistas
